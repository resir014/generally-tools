A suite of tools made for the game GeneRally by resir014.

About the Project

This is a series of web-based tools for the freeware top-down racing game GeneRally. No more installing stuff, no plugins required, just open your browser (desktop or mobile) and use it! It's also open-source so anyone can contribute to it. If you run into an issue, or if you have any tool ideas, please file an issue on GitHub or on the GR International Forum thread.

Tools

Realistic Physics Calculator

Generates realistic, scaled-down attributes for your custom GeneRally car. All values are based on Haruna's Realistic Car Physics formula with some minor adjustments. Enter the real-life attributes of a car, then click "Calculate Physics". It will then generate scaled-down attributes that you can immediately copy and paste into the GeneRally car editor.
